Betterpay payment gateway plugin for WooCommerce. 

== Description ==

Introducing Betterpay, a versatile payment gateway plugin designed for WooCommerce. Empower your customers to make payments seamlessly using Online Banking, Credit/Debit Cards, e-Wallets, and BNPL Instalment Plans. With support for a wide array of payment methods and currencies, including flexible rates sourced from reputable financial institutions across the ASEAN region. Explore our website at https://www.betterpay.me/ for hassle-free account registration and start maximizing your online transactions today.

== Installation ==

1. Ensure that you have the WooCommerce plugin installed and activated on your WordPress website.
2. Navigate to your WordPress admin dashboard and click on the 'Plugins' menu, then select 'Add New'.
3. Enter 'Betterpay' into the 'Search Plugins' field and hit enter.
4. Locate the Betterpay plugin from the search results and click on the 'Install' button.
5. After installation, activate the plugin through the 'Plugins' screen in your WordPress dashboard.
6. Proceed to the 'WooCommerce' menu, then select 'Settings', followed by 'Checkout', and choose 'Betterpay'.
7. Input your Merchant ID and API key, which can be obtained from the 
Betterpay Dashboard at https://www.Betterpay.me/merchant/.
8. Ensure that the 'Enable this payment gateway' option is checked, then click the 'Save changes' button.
9. Within the Betterpay Dashboard, enter your website URL and optionally set a return URL, then save your settings.
